# CODE CHALLENGE TEST

## Install dependencies

```bash
pnpm install
```

## Run the problem-{number} folder app

```bash
pnpm --filter problem-{number} run dev
```

- Example:

```bash
pnpm --filter problem-1 run dev
```

## Formatting

```bash
pnpm -w format
```
